Weighted 2-Limits

Let W:D→Cat and F:D→Topos. Let L be W-weighted limit of F.

Identify toposes with the theories they classify.

Weighted limits are defined by the the condition that Topos(E,L)≅[D,Cat](W,Topos(E,F−)). This means L classifies pseudonatural transformations from W to the 2-functor mapping d to the category of models of Fd. This theory is very complicated, but it is definable in geometric logic, so we have the required functoriality over E.

Explicitly, treating F as a 2-functor returning categories of models:

  • For each d∈D and w∈W(d), an object ϕ(d,w) of Fd.
  • For each d∈D and f:w1​W(d)​w2​, a model homomorphism ϕ(d,f):ϕ(d,w1​)→ϕ(d,w2​).
  • For each δ:d1​D​d2​, and w∈W(d1​), an isomorphism ϕ(δ,w):ϕ(d2​,W(δ)(w))⇔F(δ)(ϕ(d1​,w)).
  • For each d∈D and w∈W(d), an equation ϕ(d,1w​)=1ϕ(d,w)​.
  • For each d∈D, f:w1​W(d)​w2​, and g:w2​W(d)​w3​, an equation ϕ(d,g∘f)=ϕ(d,g)∘ϕ(f).
  • For each δ:d1​D​d2​, and f:w1​W(d)​w2​, an equation ϕ(δ,w2​)∘ϕ(d1​,f) = ϕ(d2​,f)∘ϕ(δ,w1​).
  • For every α:δ1​d1​D​d2​​δ2​, and w∈W(d1​), an equation ϕ(δ2​,w)∘ϕ(d2​,W(α)w​)=F(α)ϕ(d1​,w)​∘ϕ(δ1​,w).
  • If D isn't a strict 2-category, even more coherences!

Unweighted

Suppose W(d)=1 for all d. Then the list of conditions simplifies to the following:

  • For each d∈D, an object ϕ(d) of Fd.
  • For each δ:d1​D​d2​, an isomorphism ϕ(δ):ϕ(d2​)⇔F(δ)(ϕ(d1​)).
  • For each δ:d1​D​d2​, an equation ϕ(δ)∘ϕ(d1​) = ϕ(d2​)∘ϕ(δ).
  • For every α:δ1​d1​D​d2​​δ2​, an equation ϕ(δ2​)∘ϕ(d2​)=F(α)ϕ(d1​)​∘ϕ(δ1​).
  • If D isn't a strict 2-category, more coherences.

Products

Now suppose D is the discrete 2-category on a set. All conditions disappear except the first. So the theory classified by a product of toposes is simply the concatenation of the individual theories.

Is nLab wrong?

At the time of writing (Jun 26, 2024), the nLab claims that "The 2-category Topos is not all that well-endowed with limits". However, this analysis suggests that Topos should have all small limits. And the book Higher Topos Theory, Corollary 6.3.4.7, says the same holds for ∞-topoi.
